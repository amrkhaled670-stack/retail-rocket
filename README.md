​🚀 RetailRocket: Inventory & POS System (PWA)
​نظام متكامل لإدارة المخازن والمبيعات (Point of Sale) مصمم خصيصاً للمحلات والأنشطة التجارية الصغيرة والمتوسطة، مع واجهة مستخدم احترافية تدعم العمل على جميع الأجهزة.
​✨ المميزات الرئيسية (Core Features)
​📊 لوحة تحكم ذكية (Dashboard): عرض فوري لإحصائيات المبيعات، الأرباح، وتنبيهات بنقص المخزون.
​📦 إدارة المخازن (Inventory): إضافة وتعديل المنتجات مع تتبع دقيق للكميات وأسعار الشراء والبيع.
​🛒 نظام كاشير (Smart POS): واجهة بيع سريعة تدعم البحث عن المنتجات، حساب الإجمالي، وطباعة/تتبع العمليات.
​📈 تقارير الأرباح: سجل كامل لعمليات البيع يوضح المكاسب الصافية لكل حركة بيع.
​📱 PWA Ready: تطبيق ويب تقدمي يمكن تثبيته على الموبايل والعمل به "أوفلاين" في حالة انقطاع الإنترنت.
​🛠 التكنولوجيا المستخدمة (Tech Stack)
​Frontend: React.js & Vite
​Styling: Tailwind CSS (Dark Mode Optimized)
​State Management: React Hooks
​Persistence: LocalStorage / IndexedDB for data safety.
​💡 الهدف من المشروع:
​استبدال الدفاتر الورقية والأنظمة المعقدة بسيستم بسيط، سريع، وفعال يساعد صاحب العمل على مراقبة مخزنه وأرباحه من أي مكان.
